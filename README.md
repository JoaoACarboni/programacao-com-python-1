## ▶️ Como executar os códigos

1. Certifique-se de ter o [Python 3](https://www.python.org/downloads/) instalado na sua máquina.
2. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/programacao-com-python-i.git
   ```
3. Acesse a pasta do conteúdo desejado:
   ```bash
   cd "Programação com Python I/(1) Soma de Dois Números"
   ```
4. Execute o arquivo Python correspondente:
   ```bash
   python nome_do_arquivo.py
   ```
